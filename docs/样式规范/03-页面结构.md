# 页面结构

## 推荐结构顺序
- 头部卡片：标题、说明、主操作按钮
- 查询卡片：筛选条件与查询行为
- 数据卡片：表格与分页

## 详情页例外
- 对执行、发布、巡检、排障类页面，允许使用“驾驶舱式详情页”，不强制套用“查询卡片 + 数据卡片”的列表结构。
- 驾驶舱式详情页规范详见：`/Users/lingyunxieqing/Desktop/gos/docs/样式规范/13-详情驾驶舱.md`
- 这类页面优先突出：
  - 当前总状态
  - 当前卡点
  - 主时间线
  - 实时日志
  - 关键上下文

## 卡片样式
- 列表相关卡片统一圆角：`var(--radius-xl)`
- 头部卡片统一使用 `.page-header-card`
- 驾驶舱总览卡可使用更强视觉层级，但仍需沿用统一圆角与卡片阴影体系

## 标题体系
- 页面标题：`.page-title`
- 页面副标题：`.page-subtitle`

## 分页区
- 类名统一：`.pagination-area`
- 默认右对齐
- 小屏可根据页面需要居中
