# 查询区

## 组件组合
- `a-card + a-form(layout=inline)` 作为统一查询区结构
- 查询行为按钮固定：`查询`、`重置`

## 控件规范
- 文本查询：`a-input + allow-clear`
- 枚举查询：`a-select + allow-clear + placeholder="全部"`
- 时间/特殊查询按业务扩展，但样式节奏保持一致

## 宽度约束
- 常规下拉：`.filter-select` 或 `.filter-status-select` 使用 `140px`
- 应用选择器：`.application-select` 使用 `260px`

## 小屏处理
- 小屏下支持换行或转 `grid`
- 关键选择框在小屏建议 `width: 100%`
