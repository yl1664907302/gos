# 表格规范

## 基础约束
- 组件：`a-table`
- 必须设置：`row-key="id"`
- 必须设置横向滚动：`:scroll="{ x: ... }"`

## 列与内容
- 状态列统一 `a-tag`
- 时间列统一格式：`YYYY-MM-DD HH:mm:ss`
- 操作列统一 `a-button type="link" size="small"`
- 删除操作统一 `a-popconfirm` 二次确认

## 列宽拖拽
- 公共逻辑：`/Users/lingyunxieqing/Desktop/gos/frontend/src/composables/useResizableColumns.ts`
- 表头钩子类：`.resizable-header-cell`
- 通过 `minWidth/maxWidth/hitArea` 控制可拖拽边界

## 分页
- 分页放在表格下方 `.pagination-area`
- 常用配置：`show-size-changer`、`show-quick-jumper`
