# 表单规范

## 基础规范
- 表单组件统一：`a-form`
- 默认布局：`layout="vertical"`
- 必填项必须配置 `rules`

## 复杂表单布局
- 多列布局使用 `a-row + a-col`
- 参考基线：`/Users/lingyunxieqing/Desktop/gos/frontend/src/views/application/ApplicationForm.vue`

## 控件使用建议
- 文本：`a-input`
- 长文本：`a-textarea`
- 枚举：`a-select`
- 布尔：`a-switch`

## 操作区
- 提交/取消按钮放在底部
- 使用 `a-space` 右对齐

## 校验与提示
- 校验失败由表单项原生提示
- 提交失败展示后端返回的可读错误
