# 落地与禁用项

## 新增页面落地要求
- 必须复用全局基类：`.page-wrapper`、`.page-header-card`、`.page-title`、`.page-subtitle`
- 列表页优先复用 `a-card + a-form(inline) + a-table + a-pagination`
- 可拖拽列宽表格优先复用 `useResizableColumns`
- 同类新增/编辑弹窗遵循 `弹出表单.md` 统一规范

## 禁用项
- 禁止同层级页面出现明显不一致的卡片圆角、边框、间距
- 禁止操作列按钮风格混用导致层级混乱
- 禁止单页引入重背景、重渐变、重阴影破坏整体风格
- 禁止为局部页面写死不响应式的固定宽高
- 禁止通过局部覆盖让同类弹窗出现“越大屏越宽”的不受控行为
