# 前端 UI 样式规范目录

## 说明
- 本文档是前端开发、页面改造、组件重构前的统一读取入口。
- 本目录采用"一主题一文档"拆分方式维护。
- 规范基线：`/Users/lingyunxieqing/Desktop/gos/frontend/src` 当前实现。

## 使用约定
- 开发前先读取本文档，确认当前可用规范文档范围。
- 再根据本次需求涉及的组件类型，继续读取对应专项规范。
- 如果规范之间存在交叉，优先级按以下顺序执行：
  1. 全局基础与布局规范
  2. 页面结构与响应式规范
  3. 组件 UI 通用规范
  4. 业务组件专项规范
  5. 当前页面已经落地且被明确要求保留的既有设计语言
