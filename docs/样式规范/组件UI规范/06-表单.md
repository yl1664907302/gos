# 表单规范

## 基础规范
- 表单组件统一：`a-form`
- 默认布局：`layout="vertical"`
- 必填项必须配置 `rules`
- 弹窗内表单必须设置 `:required-mark="false"`，改用 tag 标识必填（详见按钮弹窗风格规范）

## 表单项样式
- label 统一设置：`font-size: 13px; font-weight: 700; color: #0f172a;`
- 表单项间距：`margin-bottom: 14px;`

```css
.xxx-form :deep(.ant-form-item-label > label) {
  color: #0f172a;
  font-size: 13px;
  font-weight: 700;
}

.xxx-form :deep(.ant-form-item) {
  margin-bottom: 14px;
}
```

## 复杂表单布局
- 多列布局使用 `a-row + a-col`
- 参考基线：`/Users/lingyunxieqing/Desktop/gos/frontend/src/views/application/ApplicationForm.vue`

## 控件使用建议
- 文本：`a-input`
- 长文本：`a-textarea`
- 枚举：`a-select`
- 布尔：`a-switch`

## 操作区
- 弹窗表单：提交按钮放在标题栏右侧，不使用底部 footer（详见按钮弹窗风格规范）
- 全页表单：提交/取消按钮放在底部，使用 `a-space` 右对齐

## 校验与提示
- 校验失败由表单项原生提示
- 提交失败展示后端返回的可读错误
