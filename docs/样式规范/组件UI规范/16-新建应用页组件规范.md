# 新建应用页组件规范

## 适用范围
- 页面：`/applications/new`
- 基线实现：
  - `/Users/lingyunxieqing/Desktop/gos/frontend/src/views/application/ApplicationCreateView.vue`
  - `/Users/lingyunxieqing/Desktop/gos/frontend/src/views/application/ApplicationForm.vue`

## 目标
- 固化新建应用页当前 UI 样式与位置，后续改动按本文档执行
- 与应用页（`/applications`）头部操作按钮保持同源风格
- 表单区与右侧信息区分层清晰，弱卡片、重信息

## 页面布局规范

### 1 顶层结构
- 顶层容器：`.page-wrapper`
- 标题栏：`.page-header.create-page-header`
- 内容区：`.create-layout`

### 2 栅格
- 桌面端（>1200px）：
  - `grid-template-columns: minmax(0, 1fr) minmax(260px, 320px)`
  - 左：表单主区（自适应）
  - 右：侧栏信息区（260~320px）
  - 列间距：`28px`
- 平板/移动端（<=1200px）：
  - 单列布局：`grid-template-columns: 1fr`
  - 右侧信息区下移到表单下方

## 组件位置规范

### 1 标题与操作按钮
- 标题：左侧，文本 `新建应用`
- 操作按钮：右侧，顺序固定
  1. `完成创建`
  2. `返回`
- 按钮位置与应用页头部按钮对齐，禁止放入表单内部

### 2 左侧表单分区与字段位置
- 分区标题：`基础信息`、`归属信息`、`配置属性`
- 分区标题左侧使用蓝色竖条（`4px x 24px`）
- 字段排布：
  - 基础信息：`应用名称` + `应用 Key`（`12 + 12`）
  - 归属信息：`归属项目` + `负责人` + `状态`（`8 + 8 + 8`）
  - 配置属性：`制品类型` + `语言`（`12 + 12`）
  - `代码仓库地址`：独占一行
  - `应用描述`：独占一行

### 3 右侧侧栏
- 卡片 1：`后续配置流程`（时间轴结构）
- 卡片 2：`命名规范（推荐）`
- 两卡片垂直堆叠，间距 `18px`

## 样式规范（按组件）

## 一 标题栏按钮
- 组件类：`.application-toolbar-action-btn.ant-btn`
- 关键样式：
  - 高度：`42px`
  - 圆角：`16px`
  - 内边距：`14px`
  - 字重：`600`
  - 毛玻璃：`backdrop-filter: blur(14px) saturate(135%)`
  - 边框：`1px solid rgba(255, 255, 255, 0.34)`
  - 背景：`rgba(255, 255, 255, 0.42)`
  - Hover/Focus：背景提亮到 `rgba(255, 255, 255, 0.56)`

## 二 表单输入组件（plain 模式）
- 作用范围：`.application-form-plain`
- 输入框/下拉统一规范：
  - 高度：`44px`
  - 圆角：`14px`
  - 边框：`rgba(255, 255, 255, 0.58)`
  - 背景：白色轻渐变
  - 阴影：弱内高光 + 弱外阴影
- Placeholder 颜色：`rgba(100, 116, 139, 0.72)`
- Hover/Focus：
  - 边框高亮为浅蓝
  - 背景与阴影轻增强

## 三 状态字段
- 当前控件：`a-select`
- 行内位置：归属信息第三列（`md=8`）
- 与其他选择框同高度同圆角，禁止单独做开关壳或额外卡片包裹

## 四 必填标签
- 类：`.field-required-hint`
- 样式：
  - 高度：`20px`
  - 字号：`11px`
  - 圆角：`6px`
  - 背景：浅蓝底
  - 文本：蓝色 `#2563eb`

## 五 分区标题
- 标题字号：`16px`
- 字重：`800`
- 分区间隔：上分区与下分区以 `border-top` 分隔
- 标题条样式：
  - 颜色：蓝色渐变
  - 阴影：轻蓝投影

## 六 侧栏卡片
- 类：`.create-side-card`
- 样式：
  - 内边距：`24px 22px`
  - 圆角：`24px`
  - 边框：浅蓝描边
  - 背景：浅色纵向渐变
  - 阴影：弱投影

### 侧栏流程时间轴
- 节点尺寸：`32px`
- 连线宽度：`2px`
- 节点图标容器为蓝色圆形渐变
- 文本层级：
  - 标题 `14px/700`
  - 描述 `13px`

## 间距与节奏规范
- 标题栏左右间距由页面容器统一控制
- 标题与内容区间距：按 `page-wrapper` 基线
- 主要间距：
  - 头部按钮间距：`12px`
  - 左右列间距：`28px`
  - 侧栏卡片间距：`18px`
  - 表单行 gutter：`12px`

## 响应式规范
- <=1200px：左右双栏改单栏
- <=768px：
  - 标题栏按钮允许换行
  - 按钮组左对齐（`justify-content: flex-start`）

## 变更约束
- 不允许将 `完成创建/返回` 按钮移回表单底部
- 不允许为状态字段增加独立卡片外壳
- 不允许新增厚重纯白大卡作为表单背景
- 侧栏卡片只允许浅色低对比风格，禁止深色驾驶舱风格

## 验收清单
- 标题栏按钮位置是否与应用页头部一致
- 表单字段是否按 `2列/3列/2列/独占行` 执行
- 状态是否为下拉，且无额外包裹壳
- 右侧是否为两张浅色信息卡
- 1200px 与 768px 两个断点布局是否符合预期
