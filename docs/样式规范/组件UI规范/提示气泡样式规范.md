# 提示气泡样式规范

## 适用范围
- 用于承载辅助说明、执行器说明、取值说明、同步时间、非阻断警告等信息
- 适合信息量较短但不需要常驻占位的说明
- 发布详情页中的阶段提示、管线提示、取值进度说明、日志同步提示均按本规范执行

## 交互规则
- 触发方式统一使用点击，不使用 hover 作为唯一入口
- 组件优先使用 `a-popover`
- `trigger="click"`
- `overlay-class-name="release-tip-popover"`
- 气泡位置跟随信息所在区域：
  - 标题左侧业务标签旁：使用 `placement="bottomLeft"`
  - 标题右侧 extra 或元信息旁：使用 `placement="bottomRight"`
  - 抽屉右上角操作区：使用 `placement="bottomRight"`

## 图标触发器
- 使用 `ExclamationCircleOutlined`
- 图标按钮类名统一为 `release-tip-trigger`
- 信息态追加 `release-tip-trigger-info`
- 警告态追加 `release-tip-trigger-warning`
- 错误态追加 `release-tip-trigger-error`
- 触发器必须是 `button type="button"`，并补充 `aria-label`
- 图标必须是轻量内联提示，不做成蓝色胶囊或实心按钮

```css
  max-width: 320px;

```

## 布局位置
- 表格或卡片说明：使用 `<template #title>` 将提示图标紧贴标题文字右侧，不放在正文顶部
- 阶段标题说明：放在右侧元信息旁，例如执行器、绑定名、更新时间之后
- 工具行说明：放在同组操作按钮前，不单独占一行
- 抽屉说明：放在抽屉右上角操作区，不放在正文顶部
- 多个提示图标同区域出现时，间距使用 `gap: 6px` 到 `8px`

## 推荐结构

### 卡片标题
```vue
<a-card :bordered="false">
  <template #title>
    卡片标题
    <a-popover
      trigger="click"
      placement="rightTop"
      overlay-class-name="release-tip-popover"
    >
      <template #content>
        <div class="release-tip-content">
          辅助说明内容
        </div>
      </template>
      <button
        class="release-tip-trigger release-tip-trigger-info"
        type="button"
        aria-label="查看说明"
      >
        <ExclamationCircleOutlined />
      </button>
    </a-popover>
  </template>
  <!-- 卡片正文 -->
</a-card>
```

### 弹窗表单
```vue
<a-form-item name="field">
  <template #label>
    字段标签
    <a-popover
      trigger="click"
      placement="rightTop"
      overlay-class-name="release-tip-popover"
    >
      <template #content>
        <div class="release-tip-content">
          字段辅助说明
        </div>
      </template>
      <button
        class="release-tip-trigger release-tip-trigger-info"
        type="button"
        aria-label="查看说明"
      >
        <ExclamationCircleOutlined />
      </button>
    </a-popover>
  </template>
  <a-input />
</a-form-item>
```

### 工具行
```vue
<a-space class="toolbar-actions">
  <a-popover
    trigger="click"
    placement="rightTop"
    overlay-class-name="release-tip-popover"
  >
    <template #content>
      <div class="release-tip-content">工具说明</div>
    </template>
    <button
      class="release-tip-trigger release-tip-trigger-info"
      type="button"
      aria-label="查看说明"
    >
      <ExclamationCircleOutlined />
    </button>
  </a-popover>
  <a-button>操作</a-button>
</a-space>
```

## 禁用项
- 禁止把辅助说明做成整块 `a-alert` 常驻在卡片正文顶部
- 禁止使用蓝色胶囊大按钮展示提示图标
- 禁止让提示图标挤占主标题视觉中心
- 禁止只用 hover 展示关键信息
- 禁止提示文案写成长段落；超过 3 行应改用抽屉或详情区
- 禁止在同一区域同时保留块状提示和气泡提示

## 实施注意事项
- 提示内容中如包含中文引号（`""`），不要让其紧邻 HTML 属性引号，避免编辑工具将属性引号转成弯引号（Unicode `""` / `""`）导致 Vue 模板解析失败
- 改造 `a-alert` 为点击式提示气泡时，卡片场景使用 `<template #title>` 将提示图标紧贴标题文字右侧，弹窗场景放在 `<a-form-item>` 的 `#label` 插槽内
- 全局样式 `release-tip-trigger` / `release-tip-content` 已在 `src/style.css` 中定义，各视图可直接使用，无需在 scoped 样式中重复声明
