# GOS Release 冒烟测试清单（2026-04-07）

## 1. 使用说明
本清单用于版本发布后、提测前、关键修复后进行快速回归。

建议执行顺序：
1. 先校验基础可访问性
2. 再校验发布主链路
3. 再校验审批、Hook、Agent、并发等增强能力

建议结论字段：
- 通过
- 失败
- 阻塞
- 不适用
