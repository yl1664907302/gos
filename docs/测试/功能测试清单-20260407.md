# GOS Release 功能测试清单（2026-04-07）

## 1. 目标
本清单用于覆盖当前 `GOS Release` 已落地的核心能力，适用于：
- 手工功能回归
- 提测前自查
- 发布后冒烟
- 版本迭代回归

本清单基于当前项目已实现功能整理，覆盖：
- 官网与登录
- 应用与标准字库
- Jenkins / ArgoCD / GitOps / Agent 组件管理
- Agent 脚本管理 / 任务管理 / 常驻任务
- 发布模板
- 发布单创建 / 审批 / 执行 / 回滚 / 重放
- 并发发布 / 排队 / 锁 / Hook
- 权限与异常场景

---

## 2. 测试范围
### 2.1 在范围内
- 前端页面功能与交互
- 后端业务流程联动
- 审批流、执行流、Hook、Agent 任务链路
- 权限与角色差异
- 关键异常与兜底行为

### 2.2 不在本轮重点范围内
- 极限性能压测
- 海量并发压测
- 长周期稳定性压测
- 第三方外部系统真实生产变更

## 6. 执行规则说明
### 6.1 当前权限口径（务必按此验证）
- 应用权限只负责：`展示`、`发起`
- 发布执行只负责：`谁发起，谁执行`
- 审批人负责：`审批`，不是默认执行人
- 发起人 = 审批人且审批人仅自己时：自动审批通过

### 6.2 当前并发口径
- 同 `app + env` 执行时需要互斥
- 同批次同 `app + env`：一张执行，其余排队
- 不同 `app` 可以并行

### 6.3 当前 Agent 口径
- Agent 主动心跳
- Agent 主动拉任务
- 常驻任务持续轮询执行
- 临时任务只执行一次
