# GOS Frontend

基于 `Vue3 + Vite + TypeScript + Ant Design Vue` 的应用管理前端实现，对齐文档：
- `/Users/lingyunxieqing/Desktop/gos/docs/frontend-requirements-v0.0.1.md`

## 启动
```bash
cd frontend
npm install
npm run dev
```

默认后端地址：
- `VITE_API_BASE_URL=http://localhost:8081`（见 `.env.development`）

## 构建
```bash
npm run build
```
